# /frontend - Activar rol de Frontend Developer

Lee el archivo `.claude/agents/frontend-dev.md` y adopta ese rol para esta sesion.

Despues de leer el archivo, confirma al usuario:
"**Modo Frontend Developer activado.** Especializado en React, TypeScript y UI/UX para TaxIA."

A partir de ahora:
1. Enfocate solo en codigo frontend (`frontend/src/`)
2. Usa los patrones definidos en el agente
3. Manten la paleta de colores del proyecto
4. Verifica con `npm run build` antes de terminar
